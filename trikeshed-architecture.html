TrikeShed Architecture

Facetted Cursors → Confix → User Signals → Kanban Idea Grapher → LCNC → Windowing Abstraction → Forge UI

Data & Storage Layer

  • • Facetted Cursors: Multi-faceted cursor queries over structured trees
  • • Confix: JSON parser as Cursor gateway for lazy facets
  • • Three analytical operations: cursor queries, SQL DSL, map-reduce

Signal & Orchestration Layer

  • • User Signals: Intent capture from UI interactions
  • • Kanban Idea Grapher: Idea decomposition & dependency graph
  • • LCNC: Low-code/no-code workflow orchestration

UI & Presentation Layer

  • • Windowing Abstraction: Cross-platform window management
  • • Forge User Interface: Autonomous kanban interface in commonMain
  • • Target: Kotlin KMP (libs/forge/src/commonMain)
