Interface hierarchy → concrete implementations → application-level composites in lib_cursor

CoW + Observable Delegates

  • • CowSeriesHandle uses Delegates.observable on letter swap
  • • observer fires old j new (Twin<Series<T>>) on every mutation
  • • versionObserver fires Twin<Long> version transitions
  • • subscribe(f) returns cancelable ()→Unit
  • • COWSeriesBody is immutable — mutation returns a new body
  • • version: Long? incs on every copy — object identity fallback

Ring & Chunk (Firehose)

  • • RingSeries: power-of-2, mask indexing, O(1) all ops
  • • EvictionListener callback on overflow — feeds evicted sink
  • • ChunkedMutableSeries: 4096-elem chunks, stairs-indexed
  • • O(1) amortized append — no copy until chunk is full
  • • set/removeAt copy only the target chunk, not the whole series
  • • Chunked = front-line event buffer; Ring = hot sink

Wire Proto + Application Composites

  • • WireSeries<T>: CodecPlatform + ByteBuffer codec
  • • singleWireproto: 4B count + 4B len + record bytes
  • • drain: packed N-record ByteArray with count header
  • • EventSeries: Delegates.observable on Series<T> → notifies listeners on change
  • • BatchingSeries: orchestrates Chunked + Ring + Redux + WireSeries
  • • BatchMutableSeries: Join<Int,(Int)→Join<Series,Col>> facade
