Blackboard Cursor-Tree · Lazy ColumnMeta Joins

()->ColumnMeta closure facade · overlay roles (OBSERVATION / AGGREGATE / ROOT) · ConfixCursor lazy row · facet dispatch

Core Type Hierarchy (Lazy Joins)

  • • Cursor = Join<Int,(Int)→RowVec> — series of RowVec, indexed by row ordinal
  • • RowVec = Join<Int,(Int)→RowVecCell> — series of cells, indexed by column ordinal
  • • RowVecCell = Join<Any?, ColumnMeta> — value + ()→ColumnMeta accessor (lazy)
  • • ColumnMeta = recursive chain: name + TypeMemento + child ColumnMeta?
  • • ColumnMetaRef adds PointcutFacet enum — runtime intent tag on each column
  • • Child rows stay lazy: facet == ChildRows → do NOT force

Blackboard Cursor-Tree + ConfixCursor

  • • top-level CRMS Blackboard Cursor — children: constants/methods/fields/edges/events/source
  • • each child = lazy ()→Cursor facade — only evaluated on access
  • • ConfixCursor: ConfixFormat (JSON/YAML/CBOR/CSV) → lazy ConfixRow sequence
  • • ConfixRow.get(idx) parses+ caches on first access; nested JSON stays raw until 2nd pass
  • • metaChain(cols): builds ColumnMeta recursively with tail=child (left-chained tree)
  • • facet(name) / join(other,on): ConfixCursor projections for column selection and faceted join

ConfixFacade · FacetReifier · Overlay Roles

  • • FacetReifier dispatch: SymbolRef→.value, Int→StringPool.resolve, Lazy→.value, MemSegment→raw
  • • walk(row) returns reified scalars + forcedAnyChild flag — does NOT force ChildRows
  • • reifyRow(row) → ReifiedRow: scalars reified, lazyForcedFlags per column
  • • EventRing: packed ring (nano/opcode/coord/symbol/wireproto), asCursor() = snapshot
  • • StringHashFacetCursor: Series<String> → hash-collapsed blackboard Cursor via poolId grouping
  • • OverlayRole: OBSERVATION (nanoTime), AGGREGATE (histogram), ROOT (parse+walk)
